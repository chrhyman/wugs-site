Welcome to the site!

If you've been dragged here against your will, it means I want to play a game with you! Due to the unfortunate reality of distances, we don't all live close to each other. But! That doesn't mean we can't still play games together! I've mentioned to some of you a few websites that I've used in the past. However, I'm also a huge fucking nerd, and I want to code my own. So here we are. You can register or, if you have already, log in. Registering just invovles setting your display name (username) and a not secure password. I will see the password you choose in plain text, so use your best judgment. (If you know anything about password security this is a huge no-no. I, however, give zero fucks.) You don't need to give an email or anything like that.

How It Works

Navigate to the games page. There, you can select from whatever number of games currently exist. I've been implementing a room system, so hopefully there can be multiple concurrent games ongoing without session data conflicting. Please let me know if you run into any issues so I can add more error handling.
